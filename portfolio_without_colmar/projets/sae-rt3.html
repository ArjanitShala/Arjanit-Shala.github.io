SAÉ RT3 – Gestion d’absences (Django)

Présentation complète du projet

Description

Dans ce projet, j’ai développé une application Django complète permettant de gérer les absences d’un département universitaire. Elle comprend des modules pour les groupes, étudiants, enseignants, cours, et absences avec un tableau de bord clair. Le système permet d'ajouter des justificatifs (PDF/image), de filtrer les absences non justifiées, et d’exporter un récapitulatif.

Ce travail m’a permis d’approfondir mes compétences en développement web avec Django : modélisation, vues CRUD, formulaires personnalisés, liens relationnels entre modèles, templates dynamiques, mise en forme CSS et déploiement dans une machine virtuelle Linux.

Durée : 5 semaines – travail en binôme

Compétences développées

  • Développement Django (models, views, templates)
  • Création d’une base relationnelle
  • Gestion de formulaires et filtres
  • Export de données

Outils et logiciels utilisés

  • Django
  • SQLite3
  • HTML/CSS
  • Python
